## Github-exam
안녕하세요 git 과제 테스트 입니다.
### Repository 테스트 항목
1. README.md 생성
2. .gitignre 생성
3. LICENSE 생성
### Github Project 칸반 테스트 항목
1. 이슈, 마일드스톤 생성
2. 생성한 이슈와 마일드스톤을 Github Project와 연결하여 칸반뷰로 변경
### Coz'Git flow 테스트 항목
1. main, dev 브랜치 각각 생성
2. dev -> main 브랜치로 Pull Request 최소 1회 시도
